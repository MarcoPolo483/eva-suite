    • Cost: $0.006/minute audio

    Recommended Approach: - Phase 1: Web Speech API (free) - Phase 2: Azure Speech with user consent (\(10-20) - Phase 3: OpenAI Whisper for multilingual (\)20-50) - Total: <$100 for 3 months

    VOICE-F-002: Text-to-Speech Response Playback - Play button reads assistant responses aloud - Waveform animation during speech playback - Pause/Resume/Stop controls - Speech rate slider (0.5x to 2x speed) - Voice selection dropdown (system voices filtered by locale) - Live region announces playback states - Word highlighting during speech

    VOICE-F-003: Voice Controls UI Component - Mic button (circle, pulsing red when recording, gray when idle) - Stop/Cancel/Play/Pause buttons with tooltips - Language switcher (EN/FR toggle with pressed state) - Speech rate slider with presets - Voice dropdown (filterable list) - Transcription display (scrollable, confidence indicators) - Keyboard accessible, focus indicators (3px gold outline) - High contrast mode support

    VOICE-F-004: Bilingual Voice Support - UI labels in EN/FR from @eva-sovereign/i18n catalogs - Live region announcements in current locale - Voice selection filtered by locale - Automatic language detection - Assistant responses in same language as query - Sample prompts in both languages

    VOICE-F-005: Accessibility Excellence (WCAG 2.1 AAA) - Skip link to voice controls - Landmark regions with aria-labels - Live regions for all state changes - Focus trap in modals - Focus management (returns to trigger after interaction) - All elements have aria-labels - Keyboard shortcuts documented - Screen reader tested (NVDA, JAWS, VoiceOver) - Color contrast ‚â•7:1 - Prefers-reduced-motion support

    VOICE-F-006: Privacy-First Architecture - Web Speech API default (local browser processing) - Privacy banner: "Your voice stays in your browser" - Cloud services opt-in checkbox - Cookie consent integration - No voice data retention - "Disable voice permanently" option - GDPR/CCPA compliance
