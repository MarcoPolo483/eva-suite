    Task 3.5: Write Migration Guide¶

    Time: 4 hours

    Create MIGRATION.md: 

    # Migration Guide

## From Lit WC to React (Option 3 Users)

### Overview
This guide helps migrate from `@eva-sovereign/web-components` (Lit) to `@eva-sovereign/react`.

### Installation
```bash
# Remove Lit WC
npm uninstall @eva-sovereign/web-components

# Install React components
npm install @eva-sovereign/react

    Component-by-Component Migration¶

    Button¶

    Before (Lit WC): 

    import '@eva-sovereign/web-components';

<eva-button variant="primary">Click</eva-button>

    After (React): 

    import { Button } from '@eva-sovereign/react';

<Button variant="primary">Click</Button>

    Input¶

    Before: 

    <eva-input
  label="Email"
  type="email"
  value={email}
  onchange={(e) => setEmail(e.target.value)}
/>

    After: 

    <Input
  label="Email"
  type="email"
  value={email}
  onChange={(e) => setEmail(e.target.value)}
/>

    Event Handling Changes¶

    Lit WC Event React Event
    onchange onChange
    onclick onClick
    onfocus onFocus
    Custom events Callback props

    Breaking Changes¶

    1. Event names now camelCase (React convention)
    2. Custom events replaced with callback props
    3. Some prop names changed for React consistency

    Migration Checklist¶

    • Update imports
    • Replace HTML tags with React components
    • Update event handlers (lowercase â†’ camelCase)
    • Test all interactive features
    • Run full test suite
    • Deploy to staging

    Task 5.2: Lighthouse Audits¶

    Time: 4 hours

    Create test/a11y/lighthouse.js: 

    import lighthouse from 'lighthouse';
import * as chromeLauncher from 'chrome-launcher';

const components = [
  'eva-button',
  'eva-input',
  // ... all 16
];

async function runLighthouse(url) {
  const chrome = await chromeLauncher.launch({ chromeFlags: ['--headless'] });
  const options = {
    logLevel: 'info',
    output: 'html',
    onlyCategories: ['accessibility', 'best-practices', 'performance', 'seo'],
    port: chrome.port
  };

  const runnerResult = await lighthouse(url, options);

  await chrome.kill();

  return runnerResult.lhr;
}

// Run for each component
for (const component of components) {
  const url = `http://localhost:6006/?path=/story/components-${component}--default`;
  const results = await runLighthouse(url);

  console.log(`${component}: Accessibility ${results.categories.accessibility.score * 100}/100`);

  if (results.categories.accessibility.score < 1.0) {
    console.error(`âŒ ${component} failed accessibility audit`);
    process.exit(1);
  }
}
