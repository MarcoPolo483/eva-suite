    Week 1 Agent 2: Implement 5 Missing GC Components¶

    Duration: 5 days (1 week) Current State: 11/16 components implemented Target: 16/16 components implemented (5 new GC components)

    üéØ Mission¶

    Implement 5 missing Government of Canada (GC) Design System compliant components to complete the EVA-Sovereign-UI foundation library.

    Missing Components: 1. <eva-gov-header> (or <gc-global-header> - verify naming) 2. <eva-gov-footer> (or <gc-global-footer> - verify naming) 3. <eva-page-shell> - Full page layout wrapper 4. <eva-hero-banner> - Hero section component 5. <eva-quick-actions-grid> - Action card grid

    üìã Component Specifications¶

    Component 1: eva-gov-header (GC Global Header)¶

    Purpose: Official Government of Canada header with wordmark, site title, language toggle, search

    GC Design System Reference: https://design.canada.ca/common-design-patterns/site-header.html

    Features: - Government of Canada wordmark (SVG) - Site title - Language toggle (EN ‚Üî FR) - Optional search bar - Skip to main content link - Mobile responsive (hamburger menu) - Breadcrumb integration - WCAG 2.2 AAA compliant

    Properties: 

    @property({ type: String }) siteTitle = '';
@property({ type: String }) currentLang = 'en';
@property({ type: Boolean }) showSearch = false;
@property({ type: String }) searchPlaceholder = 'Search Canada.ca';
@property({ type: Array }) breadcrumbs: Array<{text: string, href: string}> = [];

    Events: - language-change: Fired when language toggle clicked - search-submit: Fired when search submitted - menu-toggle: Fired when mobile menu toggled

    Template Structure: 

    <header class="gc-header">
  <div class="gc-header-container">
    <!-- Skip links -->
    <nav class="skip-links">
      <a href="#main-content">Skip to main content</a>
    </nav>

    <!-- Government of Canada wordmark -->
    <div class="gc-signature">
      <img src="..." alt="Government of Canada / Gouvernement du Canada">
    </div>

    <!-- Site title -->
    <div class="site-title">
      <slot name="title">${this.siteTitle}</slot>
    </div>

    <!-- Language toggle -->
    <div class="language-toggle">
      <button @click="${this._toggleLanguage}">
        ${this.currentLang === 'en' ? 'Fran√ßais' : 'English'}
      </button>
    </div>

    <!-- Search (optional) -->
    ${this.showSearch ? html`
      <div class="search">
        <form @submit="${this._handleSearch}">
          <input type="search" placeholder="${this.searchPlaceholder}">
          <button type="submit">Search</button>
        </form>
      </div>
    ` : ''}
  </div>

  <!-- Breadcrumbs -->
  ${this.breadcrumbs.length ? html`
    <nav aria-label="Breadcrumb">
      <ol class="breadcrumb">
        ${this.breadcrumbs.map((crumb, i) => html`
          <li>
            ${i < this.breadcrumbs.length - 1
              ? html`<a href="${crumb.href}">${crumb.text}</a>`
              : html`<span aria-current="page">${crumb.text}</span>`
            }
          </li>
        `)}
      </ol>
    </nav>
  ` : ''}
</header>

    Styles (GC Design System compliant): - Background: #26374a (GC blue) - Text: #fff - Wordmark: Official GC SVG - Font: Noto Sans, Lato, sans-serif - Mobile breakpoint: 768px

    Tests Required: - [ ] Renders with default properties - [ ] Shows/hides search based on prop - [ ] Language toggle changes lang and emits event - [ ] Search form emits event with query - [ ] Breadcrumbs render correctly - [ ] Mobile menu works - [ ] Keyboard navigation (Tab, Enter, Escape) - [ ] Screen reader announces language change - [ ] axe-core 0 violations - [ ] WCAG 2.2 AAA color contrast

    Deliverable: Fully functional GC header component with 100% test coverage

    Component 3: eva-page-shell¶

    Purpose: Complete page layout wrapper combining header, main content, footer

    Features: - Slots for header, main, footer, sidebar - Responsive layout (sidebar on desktop, stacked on mobile) - Skip to content functionality - Focus management - Landmark regions properly labeled

    Properties: 

    @property({ type: Boolean }) hasSidebar = false;
@property({ type: String }) sidebarPosition: 'left' | 'right' = 'left';
@property({ type: String }) mainId = 'main-content';
@property({ type: Boolean }) fixedHeader = false;

    Template Structure: 

    <div class="page-shell ${this.fixedHeader ? 'fixed-header' : ''}">
  <!-- Header slot -->
  <slot name="header"></slot>

  <!-- Main content area -->
  <div class="content-wrapper">
    ${this.hasSidebar && this.sidebarPosition === 'left' ? html`
      <aside class="sidebar sidebar-left">
        <slot name="sidebar"></slot>
      </aside>
    ` : ''}

    <main id="${this.mainId}" tabindex="-1">
      <slot></slot>
    </main>

    ${this.hasSidebar && this.sidebarPosition === 'right' ? html`
      <aside class="sidebar sidebar-right">
        <slot name="sidebar"></slot>
      </aside>
    ` : ''}
  </div>

  <!-- Footer slot -->
  <slot name="footer"></slot>
</div>

    Styles: - CSS Grid layout - Mobile: Single column stack - Tablet: Optional 2-column with sidebar - Desktop: Full 3-column support - Sticky header option - Main content max-width: 1200px - Responsive breakpoints: 768px, 1024px

    Tests Required: - [ ] Renders with all slots - [ ] Sidebar position works (left/right) - [ ] No sidebar mode works - [ ] Fixed header stays at top - [ ] Skip to main content focuses correctly - [ ] Responsive layout works - [ ] Keyboard navigation - [ ] Landmark regions announced - [ ] axe-core 0 violations

    Deliverable: Flexible page shell component with 100% test coverage

    Component 4: eva-hero-banner¶

    Purpose: Hero section with image, title, description, CTA buttons

    Features: - Background image or gradient - Title and description - Primary and secondary CTA buttons - Overlay for text readability - Responsive sizing - Bilingual support - WCAG 2.2 AAA contrast

    Properties: 

    @property({ type: String }) title = '';
@property({ type: String }) description = '';
@property({ type: String }) backgroundImage = '';
@property({ type: String }) backgroundColor = '#26374a';
@property({ type: String }) primaryButtonText = '';
@property({ type: String }) primaryButtonHref = '';
@property({ type: String }) secondaryButtonText = '';
@property({ type: String }) secondaryButtonHref = '';
@property({ type: String }) size: 'small' | 'medium' | 'large' = 'medium';
@property({ type: Number }) overlayOpacity = 0.5;

    Events: - primary-click: Fired when primary button clicked - secondary-click: Fired when secondary button clicked

    Template Structure: 

    <section class="hero-banner hero-${this.size}"
         style="background-color: ${this.backgroundColor}">
  ${this.backgroundImage ? html`
    <div class="hero-background"
         style="background-image: url(${this.backgroundImage})">
      <div class="hero-overlay"
           style="opacity: ${this.overlayOpacity}"></div>
    </div>
  ` : ''}

  <div class="hero-content">
    <h1 class="hero-title">${this.title}</h1>

    ${this.description ? html`
      <p class="hero-description">${this.description}</p>
    ` : ''}

    <div class="hero-actions">
      ${this.primaryButtonText ? html`
        <eva-button
          variant="primary"
          size="large"
          @click="${this._handlePrimaryClick}">
          ${this.primaryButtonText}
        </eva-button>
      ` : ''}

      ${this.secondaryButtonText ? html`
        <eva-button
          variant="secondary"
          size="large"
          @click="${this._handleSecondaryClick}">
          ${this.secondaryButtonText}
        </eva-button>
      ` : ''}
    </div>
  </div>
</section>

    Styles: - Small: 300px height - Medium: 450px height - Large: 600px height - Text overlay with 7:1 contrast - Responsive: Full height on mobile, sized on desktop - Buttons: 16px spacing

    Tests Required: - [ ] Renders with title and description - [ ] Background image displays - [ ] Background color fallback works - [ ] Primary button emits event - [ ] Secondary button emits event - [ ] Size variants work (small/medium/large) - [ ] Overlay opacity adjusts - [ ] Contrast meets WCAG 2.2 AAA - [ ] Responsive sizing works - [ ] axe-core 0 violations

  icon: string,
  title: string,
  description: string,
  href: string,
  id: string

@property({ type: Number }) columns: 1 | 2 | 3 | 4 = 3;
@property({ type: Boolean }) showIcons = true;

    Events: - action-click: Fired when action card clicked (with action details)

    Template Structure: 

    <div class="quick-actions-grid columns-${this.columns}">
  ${this.actions.map(action => html`
