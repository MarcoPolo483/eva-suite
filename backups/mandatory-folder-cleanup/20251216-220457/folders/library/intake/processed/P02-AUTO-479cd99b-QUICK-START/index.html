    üöÄ Quick Start Guide - Creating 17 EVA Repos¶

    Current Situation¶

    You have two paths to create the 17 EVA repositories. Both work, choose what's easiest for you.

    ‚ö° Option 1: GitHub CLI (RECOMMENDED - Simplest)¶

    Step 1: Install GitHub CLI¶

    Choose one method:

    Method A: Download installer 1. Visit: https://cli.github.com/ 2. Download Windows installer 3. Run installer 4. Restart your terminal (or run: refreshenv if you have Chocolatey)

    Method B: Use winget 

    winget install GitHub.cli
# Then restart terminal

    Step 2: Login to GitHub¶

    # This will open your browser - login with your GitHub account
gh auth login --web

    Follow prompts: - Choose: GitHub.com - Protocol: HTTPS - Authenticate: Login with a web browser - Copy the code shown, paste in browser

    Step 3: Create Repos¶

    cd "C:\Users\marco.presta\dev\EVA Suite\eva-orchestrator"
.\create-repos-simple.ps1

    That's it! All 17 repos will be created using your existing GitHub authentication.

    üîë Option 2: Personal Access Token (Traditional)¶

    If GitHub CLI doesn't work or you prefer tokens:

    Step 1: Create Token¶

    1. Visit: https://github.com/settings/tokens/new
    2. Note: "EVA Orchestrator - Repo Creation"
    3. Expiration: 30 days (or custom)
    4. Check these scopes:
    5. ‚úÖ repo (full control of repositories)
    6. ‚úÖ workflow (update workflows)
    7. Click "Generate token"
    8. Copy the token immediately (starts with ghp_)

    Step 2: Run Orchestrator¶

    cd "C:\Users\marco.presta\dev\EVA Suite\eva-orchestrator"

# Set token (replace with your actual token)
$env:GITHUB_TOKEN = "ghp_paste_your_token_here"

# Run orchestrator
.\run-orchestrator.ps1

    ‚ùì Troubleshooting¶

    "gh: command not found"¶

    • Restart your terminal after installing GitHub CLI
    • Or run: refreshenv (if Chocolatey installed)
    • Or manually add to PATH: C:\Program Files\GitHub CLI

    "gh not authenticated"¶

    • Run: gh auth login --web
    • Make sure to copy/paste the code in your browser

    "Build failed" error¶

    cd "C:\Users\marco.presta\dev\EVA Suite\eva-orchestrator"
npm install
npm run build

    Token expired¶

    • Create a new token: https://github.com/settings/tokens
    • Set again: $env:GITHUB_TOKEN = "ghp_new_token"
