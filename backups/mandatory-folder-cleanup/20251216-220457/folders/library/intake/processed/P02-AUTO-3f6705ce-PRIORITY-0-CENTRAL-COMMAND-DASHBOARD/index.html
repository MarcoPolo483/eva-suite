    PRIORITY 0: Central Command Dashboard for EVA Suite¶

    Date: 2025-12-08 Status: READY TO IMPLEMENT - Awaiting Marco's additional context Owner: GitHub Copilot (SM) + Marco Presta (PO) Vision: "Central Commando of the Agile Crew"

    üéØ Mission¶

    Build the nerve center for EVA Suite orchestration - a unified dashboard that eliminates slow context gathering and provides instant situational awareness across all 18 repositories, backlogs, and work streams.

    Impact: Reduce context loading from ~5 minutes (18 repo scans) to ~5 seconds (single dashboard read).

    4. Nightly Status Refresh Workflow¶

    Location: .github/workflows/status-refresh.yml Purpose: Auto-generate dashboard at 02:00 daily

    name: Status Dashboard Refresh
on:
  schedule:
    - cron: '0 2 * * *'  # 02:00 UTC daily
  workflow_dispatch:

jobs:
  refresh-dashboard:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Scan all EVA repos
        run: |
          pwsh scripts/refresh-status-dashboard.ps1

      - name: Commit dashboard
        run: |
          git config user.name "EVA Status Bot"
          git config user.email "eva-bot@github.com"
          git add status-snapshot.json
          git commit -m "chore: refresh status dashboard [skip ci]"
          git push

    5. Quick Status Command¶

    Location: eva-orchestrator/scripts/Get-EvaStatus.ps1 Purpose: On-demand status check

    <#
.SYNOPSIS
Display EVA Suite status from Central Command Dashboard

.DESCRIPTION
Reads status-snapshot.json and displays:
- Overall health
- Repos by status (production-ready, in-progress, blocked, clean)
- Critical path blockers
- Next recommended actions

.EXAMPLE
.\scripts\Get-EvaStatus.ps1

.EXAMPLE
.\scripts\Get-EvaStatus.ps1 -Detailed

.EXAMPLE
.\scripts\Get-EvaStatus.ps1 -Pod POD-F
#>

param(
    [switch]$Detailed,
    [string]$Pod
)

# Load dashboard
$dashboard = Get-Content "status-snapshot.json" | ConvertFrom-Json

# Display summary
Write-Host "`nEVA SUITE STATUS - $(Get-Date -Format 'yyyy-MM-dd HH:mm:ss')" -ForegroundColor Cyan
Write-Host "Overall Health: $($dashboard.overall_health)" -ForegroundColor Green
Write-Host "Repos: $($dashboard.summary.total_repos) total"
Write-Host "  Production Ready: $($dashboard.summary.production_ready)" -ForegroundColor Green
Write-Host "  In Progress: $($dashboard.summary.in_progress)" -ForegroundColor Yellow
Write-Host "  Blocked: $($dashboard.summary.blocked)" -ForegroundColor Red
Write-Host "  Clean: $($dashboard.summary.clean)" -ForegroundColor Gray

# Display blockers
if ($dashboard.repos.PSObject.Properties.Value.blockers.Count -gt 0) {
    Write-Host "`nCritical Blockers:" -ForegroundColor Red
    # ... display logic
}
