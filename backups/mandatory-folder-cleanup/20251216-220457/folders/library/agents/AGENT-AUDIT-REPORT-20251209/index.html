    Batch Generation Strategy¶

    Template-Based Generation (Efficient)¶

    Instead of manual creation, use P02 as template:

    # Generate 16 missing HELP files in batch
$agents = @(
    @{Id="SM"; Name="Scrum Master"; POD="POD-O"; Services="Sprint planning, backlog, ceremonies"},
    @{Id="PO"; Name="Product Owner"; POD="POD-O"; Services="Requirements, prioritization, acceptance"},
    @{Id="P01"; Name="System Architect"; POD="POD-F"; Services="Architecture, integration, decisions"},
    # ... (continue for all 16)
)

foreach ($agent in $agents) {
    .\Generate-Agent-HELP-File.ps1 `
        -AgentId $agent.Id `
        -AgentName $agent.Name `
        -Pod $agent.POD `
        -PrimaryServices $agent.Services `
        -Template "P02-DATA-ARCHITECT-HELP.md"
}

    Benefits: - Consistent structure (all match P02 quality) - Fast generation (16 files in 30 minutes) - Enforce standards from start - Marco approval once, apply to all
