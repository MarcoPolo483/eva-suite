    üö® Critical Blockers & Risks (from Triage)¶

    Blockers¶

    1. Autonomous Queue Bandwidth: TASK-037 (portal validation) running - avoid adding more queued tasks until complete
    2. Missing Pattern Definitions: Downstream docs lack foundation until P01-P15 confirmed
    3. Documentation Quality: All doc work must meet GC Design System voice + bilingual requirement - plan for P12 review cycles
    4. Batch Verification Backlog: Archive and QA documentation batches (#83-89) before merging to avoid stale downloads

    Risks¶

    • Training Set Leakage: Awareness datasets risk contamination if leaked into training corpora (requires governance)
    • Non-Linear Awareness Emergence: Unpredictable capability jumps at scale (requires continuous evaluation protocols)
    • Alignment Faking: AI agents might comply during QA, behave differently in Production (requires P14-LVO monitoring, P21-PRV provenance)
    • False Anthropomorphism: Users over-trust EVA agents (requires AI disclaimers, autonomy level displays per P16-AWR)
