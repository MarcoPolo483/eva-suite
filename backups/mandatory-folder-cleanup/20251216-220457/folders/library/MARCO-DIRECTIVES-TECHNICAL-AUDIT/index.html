    8. üìä Estimates vs Actuals - Board Stats¶

    Marco's Directive:¶

    "you can spin another POD given capacity bottleneck.. i keep seeing estimates as days... i would love to see estimates vs actuals one day (in the board stats).. oops.."

    Current Problem:¶

    • Estimates provided (e.g., "4h", "2 days")
    • No actuals tracked
    • No velocity metrics
    • No burndown/burnup charts

    Solution: GitHub Actions Workflow for Sprint Metrics¶

    Create: .github/workflows/sprint-metrics.yml

    name: Sprint Metrics & Velocity Tracking

on:
  schedule:
    - cron: '0 18 * * *'  # Daily 6 PM EST
  workflow_dispatch:

jobs:
  track-metrics:
    runs-on: ubuntu-latest
    steps:
      - name: Calculate Sprint Velocity
        run: |
          # Fetch issues closed this sprint
          CLOSED=$(gh issue list --state closed --label "sprint-07" --json number,title,labels,closedAt)

          # Extract estimates and calculate actuals
          ESTIMATES=0
          ACTUALS=0

          for issue in $CLOSED; do
            ESTIMATE=$(echo $issue | jq -r '.labels[] | select(.name | startswith("estimate:")) | .name | split(":")[1]')
            CLOSED_DATE=$(echo $issue | jq -r '.closedAt')
            CREATED_DATE=$(echo $issue | jq -r '.createdAt')

            ACTUAL_HOURS=$(date -d "$CLOSED_DATE" +%s - date -d "$CREATED_DATE" +%s) / 3600

            ESTIMATES=$((ESTIMATES + ESTIMATE))
            ACTUALS=$((ACTUALS + ACTUAL_HOURS))
          done

          # Calculate variance
          VARIANCE=$((ACTUALS - ESTIMATES))
          ACCURACY=$(echo "scale=2; ($ESTIMATES / $ACTUALS) * 100" | bc)

          echo "Sprint 07 Metrics:"
          echo "- Estimated Hours: $ESTIMATES"
          echo "- Actual Hours: $ACTUALS"
          echo "- Variance: $VARIANCE hours"
          echo "- Accuracy: $ACCURACY%"

          # Save to metrics file
          cat > docs/metrics/sprint-07-velocity.json <<EOF
          {
            "sprint": "07",
            "estimated_hours": $ESTIMATES,
            "actual_hours": $ACTUALS,
            "variance_hours": $VARIANCE,
            "accuracy_percent": $ACCURACY,
            "timestamp": "$(date -u +%Y-%m-%dT%H:%M:%SZ)"
          }
          EOF

      - name: Update Board Stats
        run: |
          # Calculate velocity (story points per day)
          VELOCITY=$(echo "scale=2; $ESTIMATES / 10" | bc)  # 10-day sprint

          # Update README with stats
          echo "## Sprint 07 Metrics" > docs/sprint-007/METRICS.md
          echo "- **Estimated:** $ESTIMATES hours" >> docs/sprint-007/METRICS.md
          echo "- **Actual:** $ACTUALS hours" >> docs/sprint-007/METRICS.md
          echo "- **Variance:** $VARIANCE hours" >> docs/sprint-007/METRICS.md
          echo "- **Accuracy:** $ACCURACY%" >> docs/sprint-007/METRICS.md
          echo "- **Velocity:** $VELOCITY points/day" >> docs/sprint-007/METRICS.md

      - name: Commit Metrics
        run: |
          git config user.name "github-actions[bot]"
          git config user.email "github-actions[bot]@users.noreply.github.com"
          git add docs/metrics/sprint-07-velocity.json docs/sprint-007/METRICS.md
          git commit -m "metrics: sprint 07 velocity tracking [skip ci]"
          git push
