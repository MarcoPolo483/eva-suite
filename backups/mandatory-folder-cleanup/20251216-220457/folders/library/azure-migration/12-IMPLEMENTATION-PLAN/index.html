    Risk Mitigation Contingency
    Azure provisioning delays Start Day 1 Use personal Azure account if corp delayed
    MAF agent complexity Focus on base class first Fallback to OpenAI SDK if MAF issues
    Demo environment failure 48-hour burn-in test Use backup video, reschedule live demo
    Cost overruns Daily monitoring, $90 alert Pause Functions App, investigate
    Integration issues Dual-write strategy Rollback to GitHub Actions
    Stakeholder availability Send invite on Day 1 Record demo, share async
