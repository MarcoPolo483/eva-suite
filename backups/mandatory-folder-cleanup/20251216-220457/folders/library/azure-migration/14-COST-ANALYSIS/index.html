    Cost Analysis - Budget & Scaling Projections¶

    Document Classification: Internal Version: 1.0 Date: December 11, 2025

    Executive Summary¶

    30-Day Demo Phase Cost¶

    • Total: $75 USD
    • Environment: DEV (Canada Central)
    • Period: December 11, 2025 - January 10, 2026

    Post-Production Annual Cost¶

    • All Environments: **\(3,900 USD/year** (\)325/month)
    • Environments: DEV + Staging + Production
    • ROI: Break-even immediate (autonomous execution value)

    DEV Environment Cost Breakdown (Monthly)¶

    Service SKU/Tier Usage Estimate Monthly Cost
    Azure AI Foundry GPT-4o (Pay-per-use) 500K tokens/month $40
    Cosmos DB Serverless 1GB storage, 10K RU/s $15
    Azure Functions Consumption Plan 200K executions, 400K GB-s $10
    Service Bus Standard 5 queues, 1M messages $5
    Application Insights Pay-as-you-go 5GB ingestion $5

    Storage Account LRS Standard 1GB storage <$1

    Cost Assumptions (DEV)¶

    • Tasks/Month: 20 tasks × 30 days ÷ 2 (every 30 min) = ~720 executions
    • Tokens/Task: ~700 tokens average (500 input + 200 output)
    • Total Tokens: 720 × 700 = ~504K tokens/month
    • GPT-4o Pricing: $2.50/1M input tokens, $10/1M output tokens
    • Input cost: (504K × 0.7) ÷ 1M × \(2.50 = ~\)0.88
    • Output cost: (504K × 0.3) ÷ 1M × \(10 = ~\)1.51
    • Total GPT-4o: ~$2.39 (budgeted $40 for headroom)

    30-Day Demo Period Detailed Costs¶

    Week-by-Week Projection¶

    Week Focus Estimated Cost Cumulative
    Week 1 (Dec 11-17) Infrastructure setup $15 $15
    Week 2 (Dec 18-24) Agent testing (low volume) $18 $33
    Week 3 (Dec 25-31) Integration testing (high volume) $22 $55
    Week 4 (Jan 1-10) Demo prep (moderate volume) $20 $75

    Cost Drivers by Service¶

    Azure AI Foundry (GPT-4o)¶

    • Week 1: ~100K tokens (setup, validation) = $0.50
    • Week 2: ~150K tokens (agent testing) = $0.75
    • Week 3: ~200K tokens (integration tests) = $1.00
    • Week 4: ~150K tokens (demo prep) = $0.75
    • Total: ~600K tokens = $3.00

    Budgeted at $40/month for safety margin (13x actual usage)

    Cosmos DB (Serverless)¶

    • Storage: 1GB (task queue + execution history) = $0.25/GB
    • Request Units: 10K RU/s average × 720 hours = ~$14.75
    • Total: $15/month

    Azure Functions (Consumption)¶

    • Executions: 720 orchestrator runs × 4 PODs = 2,880 executions
    • Execution Time: 5 min avg × 2,880 = 14,400 min = 240 hours
    • GB-seconds: 1GB RAM × 240 hours × 3,600 = 864,000 GB-s
    • Cost: (864K GB-s - 400K free) × \(0.000016 = **\)7.42**

    Budgeted at $10/month

    Service Bus (Standard)¶

    • Base: $10/month (includes 13M operations)
    • Messages: 720 executions × 5 messages (send/receive/complete/status/dlq) = 3,600 messages
    • Total: $0.00 (well within free tier)

    Budgeted at $5/month for future scaling

    Application Insights¶

    • Ingestion: 5GB/month (traces, metrics, logs)
    • Cost: 5GB × $2.30/GB (first 5GB free in many regions)
    • Total: $0.00 (free tier)

    Budgeted at $5/month for safety margin

    Cost Optimization Strategies¶

    Immediate (Demo Phase)¶

    1. Serverless Cosmos DB: Pay only for actual usage
    2. Consumption Functions: No idle cost
    3. Standard Service Bus: Avoid Premium tier
    4. Free tiers: Leverage Application Insights free 5GB

    Short-term (3-6 months)¶

    1. Reserved Instances: 30% savings on Functions Premium
    2. Cosmos DB Autoscale: Scale down during low usage
    3. Token Optimization: Reduce GPT-4o prompt sizes by 20%
    4. Caching: Cache frequent agent responses

    Long-term (6-12 months)¶

    1. Azure Savings Plan: 3-year commit for 50% discount
    2. Spot Instances: For non-critical workloads
    3. Multi-tenancy Revenue: Offset costs with PO subscriptions
    4. Custom Model Fine-tuning: Reduce token costs by 40%

    Daily Cost Report (PowerShell)¶

    # Get daily cost
$cost = az consumption usage list `
  --start-date (Get-Date).AddDays(-1).ToString("yyyy-MM-dd") `
  --end-date (Get-Date).ToString("yyyy-MM-dd") `
  --query "[].{cost:pretaxCost}" `
  --output json | ConvertFrom-Json

Write-Host "Yesterday's cost: `$$($cost.cost)" -ForegroundColor Cyan

    Application Insights Cost Query¶

    // Daily cost breakdown by service
Usage
| where TimeGenerated > ago(24h)
| summarize TotalGB = sum(Quantity) / 1024 by DataType
| extend EstimatedCost = TotalGB * 2.30
| project DataType, TotalGB, EstimatedCost
| order by EstimatedCost desc

    Cost Attribution Model (Multi-Tenant)¶

    Per-POD Cost Tracking¶

    Cosmos DB: - Partition by /pod: Exact RU consumption per POD - Query: SELECT c.pod, SUM(c._ru) FROM c GROUP BY c.pod

    Functions: - Tag each execution with POD in App Insights custom dimension - Query: customDimensions.pod

    GPT-4o: - Track tokens per task (stored in executions container) - Cost formula: (inputTokens * $0.0000025) + (outputTokens * $0.00001)

    Monthly Invoice per POD: 

    POD-F: $18.50
POD-O: $22.30
POD-Q: $19.80

    ROI Analysis¶

    Cost Avoidance¶

    Manual Execution Cost (without automation): - 20 tasks/month × 2 hours/task = 40 hours/month - Hourly rate: $75/hour (developer time) - Monthly Savings: 40 × \(75 = **\)3,000**

    Net ROI: - Savings: $3,000/month - Cost: $75/month (DEV) or \(400/month (all environments) - **ROI (DEV)**: (\)3,000 - $75) / \(75 = **3,900%** - **ROI (All Env)**: (\)3,000 - $400) / $400 = 650%

    Time to Value¶

    • Development: 30 days (Dec 11 - Jan 10)
    • First task automated: Day 15 (Dec 25)
    • Full automation: Day 30 (Jan 10)
    • Break-even: Immediate (first automated task)
