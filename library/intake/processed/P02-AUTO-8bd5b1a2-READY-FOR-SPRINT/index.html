    Step 1: Kick Off Sprint¶

    gh workflow run sprint-168h-kickoff.yml \
  -f sprint_goal="Prove autonomous 1-POD model works" \
  -f target_velocity="5" \
  -f budget_weekly="23"

    This will: - Create sprint plan using P02 + MSM analysis - Set baseline metrics (backlog count, velocity target) - Trigger all autonomous agents (MSM, PSM-1, DM-1) - Save sprint tracking to eva-lwo

    Step 2: Monitor Progress¶

    • Daily standups: eva-lwo/daily-standups/{date}.md

    • Cost tracking: eva-lwo/cost-tracking/{date}.json
    • Heartbeat logs: eva-lwo/heartbeat-logs/{date}-{time}.json

    Step 3: Deploy Spark Experts¶

    When DevMaster posts Spark prompt to issue: 1. Go to https://github.com/spark 2. Paste the prompt from issue comment 3. Review generated app 4. Click "Create Repository" 5. Expert implements solution, opens PR

    Step 4: Review & Merge¶

    • Expert creates PR to eva-orchestrator
    • QA gate runs tests automatically
    • Marco reviews and merges
    • Agent counts toward velocity
