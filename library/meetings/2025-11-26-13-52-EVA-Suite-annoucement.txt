EVA Chat Frame 0.1.0 is the **first integrated React/Vite shell** for EVA‚Äôs chat experience: bilingual, accessible, and ready to plug into EVA‚Äôs APIM-backed services. It‚Äôs a working scaffold we can now reuse for future EVA chat surfaces.

#### ‚úÖ What‚Äôs included

**1. Unified Shell & Layout**

* React + Vite app with:

  * `<header>`, `<main>`, and `<footer>` structure
  * Bilingual **skip links** (screen-reader and keyboard-friendly)
  * Global **locale selector** in the header (en-CA / fr-CA)
* Dedicated layout components:

  * `AppHeader`, `AppFooter`, `RootLayout`, `SkipLinks`

**2. Two-Pane Chat Experience**

* Left sidebar with:

  * ‚ÄúNew chat‚Äù entry point
  * Space for future thread list
* Main chat area with:

  * Scrollable message history (`role="log"` + `aria-live`)
  * Localized chat bubbles for user/assistant
  * `ChatInputBar` with:

    * **Enter** to send
    * **Shift+Enter** for newline
  * **Mic button (simulated)** that inserts a sample dictated message
  * **Mode toggle** (Internal / External) with telemetry events

**3. Minimal EVA APIM Client & Telemetry**

* `evaApiClient` stub accepting:

  * `x-project`, `x-feature`, `x-app`, `x-env`, `x-cost-center`, `x-mode`, `x-locale`
* Returns mock responses that reflect:

  * The active **mode** (Internal/External)
  * The active **locale**
* All send/receive/mode/locale events are logged through `telemetry.ts`, paving the way for FinOps and observability later.

**4. Bilingual i18n & Branding**

* Centralized runtime i18n store (`i18n.ts`)
* Locale bundles:

  * `public/locale/en-CA.json`
  * `public/locale/fr-CA.json`
* Every UI string is localized:

  * Skip links
  * Header/footer copy
  * Sidebar labels and buttons
  * Placeholders and error messages
* EVA branding:

  * `eva-logo.svg`
  * Updated favicon

**5. Accessibility & Quality Checks**

* Skip links target main content and chat input.
* Focus outlines preserved (no ‚Äúoutline: none‚Äù hacks).
* Landmarks and roles set for screen readers.
* `npm run lint` (TypeScript type-check) passes.
