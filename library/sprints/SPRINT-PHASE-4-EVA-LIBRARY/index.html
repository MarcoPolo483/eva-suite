    • Each Book generates manifest.json + chunked .txt files
    • RBAC metadata: Privacy=public, Roles=[Anonymous, Authenticated]
    • Citation objects: {bookTitle, chapterTitle, chapterUrl, excerpt}
    • Output location: site/rag-output/ (deployed with GitHub Pages)
