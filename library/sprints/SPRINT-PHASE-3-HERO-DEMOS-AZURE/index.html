    Sprint Backlog: EVA Suite Portal - Phase 3 (Hero Demos Azure Deployment)¶

    Sprint: Phase 3 (Weeks 3-6)
    Objective: Deploy 6 hero demos to Azure Static Web Apps and integrate launchers into EVA Suite Portal
    Related: REQ-EVA-Suite-Portal.md Phase 3, AZURE-HERO-DEMO-DEPLOYMENT.md

    Epic: EPIC-003 - Hero Demos Azure Deployment¶

    Description: Deploy 6 hero demos to Azure Static Web Apps (separate from portal) and create launcher integration in EVA Suite Portal

    Sprint Capacity: 4 weeks (80 hours per developer)
    Team: POD-X (UI Pod)
    Priority: HIGH (Q1 2026 graduations depend on this)

    Week 3 Tasks (EVA DA Demo + LiveOps Demo Setup)¶

    TASK-301: Create EVA DA Demo Repo and Azure Resource¶

    Story Points: 5
    Assignee: POD-X
    Description: Bootstrap standalone repo for EVA DA demo and provision Azure Static Web App

    Acceptance Criteria: - [ ] Create repo: eva-da-demo (or eva-da if graduating soon) - [ ] Initialize with Vite + React 19 + TypeScript - [ ] Add GC Design System components - [ ] Copy demo code from archive (if exists) or scaffold new - [ ] Provision Azure Static Web App: eva-da-dev-swa - [ ] Configure DNS: https://eva-da-dev.z9.web.core.windows.net/ - [ ] Add GitHub Secret: AZURE_STATIC_WEB_APPS_API_TOKEN_DEV

    Subtasks: 1. Create GitHub repo MarcoPolo483/eva-da-demo 2. Bootstrap with npm create vite@latest eva-da-demo -- --template react-ts 3. Install dependencies: @eva/sovereign-ui, react-router-dom@7 4. Azure Portal: Create Static Web App resource 5. Add deployment token to GitHub Secrets 6. Test deployment manually

    Related Files: - Template: templates/github-actions/azure-static-web-app-deploy.yml - Naming: eva-da-dev-swa

    TASK-302: Implement EVA DA Demo Features¶

    Story Points: 8
    Assignee: POD-X
    Description: Build interactive EVA DA demo (document upload, semantic search, RAG citations)

    Acceptance Criteria: - [ ] Document upload UI (drag-drop or file picker) - [ ] Semantic search input with autocomplete - [ ] RAG results with citation cards (source, confidence, snippet) - [ ] RBAC-aware UI (demo mode, synthetic data) - [ ] Bilingual EN-CA/FR-CA - [ ] WCAG 2.1 AA compliant - [ ] Synthetic demo data (10+ documents, 50+ citations)

    Subtasks: 1. Create DocumentUpload component (drag-drop zone) 2. Create SemanticSearch component (search bar + autocomplete) 3. Create CitationCard component (GC citation format) 4. Add mock RAG service (synthetic responses) 5. Add i18n translations (en/fr) 6. Add accessibility tests (jest-axe)

    Related: - Requirements: REQ-EVA-Suite-Portal.md STORY-003 - Component Spec: HERO-DEMO-LAUNCHER-SPEC.md

    TASK-303: Deploy EVA DA Demo to Azure¶

    Story Points: 3
    Assignee: POD-X
    Description: Configure GitHub Actions workflow and deploy to Azure Static Web App

    Acceptance Criteria: - [ ] Copy workflow template to .github/workflows/deploy-dev.yml - [ ] Replace {PRODUCT} placeholders with eva-da - [ ] Update build settings (output_location: "dist") - [ ] Test deployment pipeline - [ ] Verify URL: https://eva-da-dev.z9.web.core.windows.net/ - [ ] Validate SSL certificate - [ ] Test demo functionality on deployed URL

    Subtasks: 1. Copy templates/github-actions/azure-static-web-app-deploy.yml 2. Customize for eva-da (replace placeholders) 3. Commit and push to trigger workflow 4. Monitor GitHub Actions run 5. Smoke test deployed demo

    Related Files: - Template: templates/github-actions/azure-static-web-app-deploy.yml - Workflow: eva-da-demo/.github/workflows/deploy-dev.yml
