    What is Context Engineering?¶

    Context Engineering ensures AI-generated code: - ‚úÖ Aligns with EVA's mission and constraints (WCAG 2.1 AA, GC Design System, bilingualism, RBAC, Protected B) - ‚úÖ Follows existing specs and contracts (not invented behavior) - ‚úÖ Reuses existing components/patterns (no reinventing wheels) - ‚úÖ Includes self-review against requirements
