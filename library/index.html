    EVA Suite Documentation Portal¶

    Welcome to the EVA Suite - Enterprise-grade AI assistant platform for the Government of Canada.

    ğŸ¯ What is EVA?¶

    EVA (Enterprise Virtual Assistant) is a cloud-agnostic AI platform designed for Protected B workloads, providing:

    • RAG-powered chat assistants with domain-specific knowledge
    • Autonomous agents following GC Agentic AI Framework (C0-C3 autonomy levels)
    • Production-ready UI components (WCAG 2.2 AAA, bilingual EN-CA/FR-CA)
    • Complete SDLC automation (P01-P21 DevTools patterns)
    • LiveOps dashboards (Audit, FinOps, Compliance)
