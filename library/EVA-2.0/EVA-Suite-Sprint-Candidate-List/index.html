    EVA Suite Sprint Candidate List¶

    Purpose: Prioritized list of sprint-ready features from P02 CDD processing
    Owner: EVA Agile Command (Scrum Master)
    Created: 2025-11-25
    Status: Ready for Sprint Planning

    Executive Summary¶

    Total Candidates: 10 requirements (UI, I18N, I11Y, I11Y-DEMO, DOMAIN-CORE, CHAT-SERVICE, AUDIT-TRAIL, PORTAL, ANALYTICS, LIVEOPS)
    Total Estimated Effort: 548-556 hours across 11+ sprints
    Critical Path: PREQ-I18N Phase 1 (Sprint 05, 16h) → PREQ-DOMAIN-CORE + PREQ-CHAT-SERVICE (Sprint 06, 44h parallel) → PREQ-PORTAL Phase 1 (Sprint 07, 32h) → PREQ-ANALYTICS + PREQ-LIVEOPS (Sprint 08, 56h parallel)
    Immediate Priority: Sprint 05 – Foundational libraries (36-44h), Sprint 06 – Critical infrastructure (56h), Sprint 07-11 – Platform expansion (Portal, Analytics, LiveOps)

    New Critical Features (2025-11-25 Intake Batch 1): - PREQ-DOMAIN-CORE (eva-rag, 88h total): RAG engine for conversational knowledge - PREQ-CHAT-SERVICE (eva-chat, 80h total): General-purpose conversational AI assistant - PREQ-AUDIT-TRAIL (eva-orchestrator, 68h total): File change tracking + compliance logging (Phase 1 complete ✅)

    New Phase 1 Requirements (2025-11-25 Intake Batch 2): - PREQ-PORTAL (eva-portal, 128h total): Landing page + design system + SSO (Sprint 07-10) - PREQ-ANALYTICS (eva-analytics, 108h total): Dashboard suite (Command Center, Usage, Learning, Space) (Sprint 08-11) - PREQ-LIVEOPS (eva-ops, 72h total): Operations management platform (Command Center + Alerting) (Sprint 08-11)
