    Chapter 08: Theming¶

    EVA Sovereign UI uses CSS Custom Properties (design tokens) to provide a flexible theming system while maintaining compliance with the Canada.ca Design System. The canada-gc theme ships as the default and enforces GC branding guidelines.

    Design Tokens Architecture¶

    What are Design Tokens?¶

    Design tokens are named variables that store visual design attributes (colors, typography, spacing, shadows). They enable consistent styling across components and simplify theme customization.

    EVA Sovereign UI exposes over 100 design tokens as CSS custom properties:

    :root {
  /* Colors */
  --gc-color-primary: #26374A;
  --gc-color-secondary: #EE7100;
  --gc-color-text: #333333;
  --gc-color-background: #FFFFFF;

  /* Typography */
  --gc-font-family: 'Noto Sans', sans-serif;
  --gc-font-size-base: 16px;
  --gc-line-height-base: 1.5;

  /* Spacing */
  --gc-spacing-xs: 0.25rem;
  --gc-spacing-sm: 0.5rem;
  --gc-spacing-md: 1rem;
  --gc-spacing-lg: 1.5rem;
  --gc-spacing-xl: 2rem;

  /* Shadows */
  --gc-shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.12);
  --gc-shadow-md: 0 4px 6px rgba(0, 0, 0, 0.15);
  --gc-shadow-lg: 0 10px 20px rgba(0, 0, 0, 0.18);
}

    Token Categories¶

    EVA Sovereign UI organizes tokens into semantic categories:

    1. Color Tokens (--gc-color-*) - Primary, secondary, accent colors - Text colors (primary, secondary, muted) - Background colors (surface, elevated, overlay) - Semantic colors (success, warning, error, info) - Border colors

    2. Typography Tokens (--gc-font-*, --gc-text-*) - Font families (primary, monospace) - Font sizes (xs, sm, md, lg, xl, 2xl, 3xl) - Font weights (light, regular, medium, bold) - Line heights - Letter spacing

    3. Spacing Tokens (--gc-spacing-*) - Margin/padding scale (xs, sm, md, lg, xl, 2xl, 3xl) - Gap values for flexbox/grid

    4. Border Tokens (--gc-border-*) - Border widths - Border radius values - Border colors

    5. Shadow Tokens (--gc-shadow-*) - Elevation shadows (sm, md, lg, xl) - Focus ring shadows

    6. Animation Tokens (--gc-duration-*, --gc-easing-*) - Transition durations (fast, normal, slow) - Easing functions (ease-in, ease-out, ease-in-out)

    Dark Mode¶

    EVA Sovereign UI supports dark mode through a prefers-color-scheme media query and manual theme toggle.

    Automatic Dark Mode¶

    :root {
  --gc-color-background: #FFFFFF;
  --gc-color-text-primary: #333333;
}

@media (prefers-color-scheme: dark) {
  :root {
    --gc-color-background: #1A1A1A;
    --gc-color-text-primary: #E0E0E0;
    --gc-color-text-secondary: #B0B0B0;
    --gc-color-border: #3A3A3A;

    /* Adjust shadows for dark backgrounds */
    --gc-shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.5);
    --gc-shadow-md: 0 4px 6px rgba(0, 0, 0, 0.6);
  }
}

    Manual Dark Mode Toggle¶

    <eva-gc-theme-toggle></eva-gc-theme-toggle>

<script>
  import { ThemeService } from '@eva-suite/sovereign-ui';

  // Get current theme
  const currentTheme = ThemeService.getTheme(); // 'light' or 'dark'

  // Set theme manually
  ThemeService.setTheme('dark');

  // Toggle theme
  ThemeService.toggleTheme();

  // Listen for theme changes
  ThemeService.onThemeChange((newTheme) => {
    console.log(`Theme changed to ${newTheme}`);

    // Update body class
    document.body.classList.toggle('theme-dark', newTheme === 'dark');

    // Persist preference
    localStorage.setItem('theme-preference', newTheme);
  });
</script>

    Custom Theme Creation¶

    Create custom themes while maintaining GC Design System compliance.

    Theme Definition¶

    /* custom-theme.css */
:root {
  /* Override GC tokens with custom values */
  --gc-color-primary: #1B4F72;         /* Custom navy */
  --gc-color-secondary: #D68910;       /* Custom gold */

  /* Custom spacing scale (8px grid instead of 4px) */
  --gc-spacing-xs: 0.5rem;    /* 8px */
  --gc-spacing-sm: 1rem;      /* 16px */
  --gc-spacing-md: 1.5rem;    /* 24px */
  --gc-spacing-lg: 2rem;      /* 32px */
  --gc-spacing-xl: 3rem;      /* 48px */

  /* Custom typography */
  --gc-font-family: 'Inter', sans-serif;
  --gc-font-size-base: 18px;  /* Larger base size */
}

    Programmatic Theme Application¶

    import { ThemeService } from '@eva-suite/sovereign-ui';

// Define custom theme
const customTheme = {
  colors: {
    primary: '#1B4F72',
    secondary: '#D68910',
    background: '#FAFAFA',
    text: '#2C3E50'
  },
  spacing: {
    xs: '0.5rem',
    sm: '1rem',
    md: '1.5rem',
    lg: '2rem',
    xl: '3rem'
  },
  typography: {
    fontFamily: 'Inter, sans-serif',
    fontSize: '18px',
    lineHeight: '1.6'
  }
};

// Apply custom theme
ThemeService.applyTheme(customTheme);

// Or apply as CSS custom properties
function applyCustomTheme(theme: typeof customTheme) {
  const root = document.documentElement;

  // Colors
  root.style.setProperty('--gc-color-primary', theme.colors.primary);
  root.style.setProperty('--gc-color-secondary', theme.colors.secondary);
  root.style.setProperty('--gc-color-background', theme.colors.background);
  root.style.setProperty('--gc-color-text', theme.colors.text);

  // Spacing
  root.style.setProperty('--gc-spacing-xs', theme.spacing.xs);
  root.style.setProperty('--gc-spacing-sm', theme.spacing.sm);
  root.style.setProperty('--gc-spacing-md', theme.spacing.md);
  root.style.setProperty('--gc-spacing-lg', theme.spacing.lg);
  root.style.setProperty('--gc-spacing-xl', theme.spacing.xl);

  // Typography
  root.style.setProperty('--gc-font-family', theme.typography.fontFamily);
  root.style.setProperty('--gc-font-size-base', theme.typography.fontSize);
  root.style.setProperty('--gc-line-height-base', theme.typography.lineHeight);
}

applyCustomTheme(customTheme);

    Theme Validation¶

    Ensure custom themes maintain accessibility:

    interface ThemeValidationResult {
  valid: boolean;
  errors: string[];
  warnings: string[];
}

function validateTheme(theme: any): ThemeValidationResult {
  const errors: string[] = [];
  const warnings: string[] = [];

  // Validate color contrast
  const textContrast = calculateContrastRatio(
    theme.colors.text,
    theme.colors.background
  );

  if (textContrast < 4.5) {
    errors.push(`Text contrast ratio ${textContrast} is below WCAG AA minimum (4.5:1)`);
  }

  // Validate font size
  const fontSize = parseInt(theme.typography.fontSize);
  if (fontSize < 14) {
    warnings.push(`Base font size ${fontSize}px is below recommended minimum (14px)`);
  }

  // Validate required tokens
  const requiredTokens = ['primary', 'secondary', 'background', 'text'];
  for (const token of requiredTokens) {
    if (!theme.colors[token]) {
      errors.push(`Missing required color token: ${token}`);
    }
  }

  return {
    valid: errors.length === 0,
    errors,
    warnings
  };
}

// Usage
const validation = validateTheme(customTheme);
if (!validation.valid) {
  console.error('Theme validation failed:', validation.errors);
} else if (validation.warnings.length > 0) {
  console.warn('Theme warnings:', validation.warnings);
}

    Theme Switching¶

    Support multiple themes in a single application.

    Theme Registry¶

    import { ThemeService } from '@eva-suite/sovereign-ui';

// Register multiple themes
ThemeService.registerTheme('canada-gc', {
  name: 'Canada.ca',
  tokens: { /* GC Design System tokens */ }
});

ThemeService.registerTheme('high-contrast', {
  name: 'High Contrast',
  tokens: {
    colors: {
      primary: '#000000',
      background: '#FFFFFF',
      text: '#000000',
      border: '#000000'
    }
  }
});

ThemeService.registerTheme('large-text', {
  name: 'Large Text',
  tokens: {
    typography: {
      fontSize: '20px',
      lineHeight: '1.8'
    },
    spacing: {
      xs: '0.5rem',
      sm: '1rem',
      md: '2rem',
      lg: '3rem',
      xl: '4rem'
    }
  }
});

// Switch between themes
ThemeService.setActiveTheme('high-contrast');

    Theme Selector Component¶

    <eva-gc-select
  id="themeSelector"
  label-en="Theme"
  label-fr="Thme"
>
  <option value="canada-gc">Canada.ca (Default)</option>
  <option value="high-contrast">High Contrast</option>
  <option value="large-text">Large Text</option>
  <option value="dark">Dark Mode</option>
</eva-gc-select>

<script>
  const selector = document.getElementById('themeSelector');

  selector.addEventListener('eva-change', (event) => {
    const themeName = event.detail.value;
    ThemeService.setActiveTheme(themeName);

    // Persist preference
    localStorage.setItem('theme-preference', themeName);
  });

  // Restore saved theme
  const savedTheme = localStorage.getItem('theme-preference');
  if (savedTheme) {
    ThemeService.setActiveTheme(savedTheme);
    selector.value = savedTheme;
  }
</script>

sd.buildAllPlatforms();

    Summary¶

    EVA Sovereign UI's theming system provides:

    • Design tokens: 100+ CSS custom properties for consistent styling
    • GC Design System compliance: Official canada-gc theme enforces GC branding
    • Dark mode support: Automatic and manual theme switching
    • Custom themes: Override tokens while maintaining accessibility
    • Component-level theming: Granular control over individual components
    • Design tool integration: Export tokens for Figma, Sketch, Adobe XD

    All themes enforce WCAG 2.1 AA color contrast requirements and maintain bilingual support.
