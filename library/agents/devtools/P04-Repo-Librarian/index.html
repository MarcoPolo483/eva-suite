    Pattern P04: Repo Librarian¶

    Status: Draft Owner: POD-F (Foundation) Version: 1.0.0 Last Updated: 2025-12-12

    Overview¶

    The Repo Librarian syncs README and ADR (Architecture Decision Record) coverage across EVA Suite repositories, recording doc drift and fixes to maintain documentation quality and consistency.

    Purpose¶

    • Maintain documentation freshness across all repos
    • Detect doc drift between code and documentation
    • Enforce consistent README structure
    • Track ADR coverage for architectural decisions
    • Enable bilingual documentation compliance

    Repository Scope¶

    Target Repositories (10 core repos)¶

    1. eva-orchestrator - SDLC automation
    2. eva-api - API gateway
    3. eva-core - Domain models
    4. eva-rag - RAG engine
    5. eva-infra - Infrastructure as code
    6. eva-sovereign-ui - UI component library
    7. eva-meta - Governance registry
    8. eva-mcp - Protocol bridges
    9. eva-auth - Authentication services
    10. eva-matrix - Public documentation

    Doc Drift Detection¶

    Automated Checks¶

    Script: tools/librarian/check-drift.ps1

    # Check documentation drift across repos
param(
    [string[]]$Repos = @(
        "eva-orchestrator",
        "eva-api",
        "eva-core",
        "eva-rag",
        "eva-infra",
        "eva-sovereign-ui",
        "eva-meta",
        "eva-mcp",
        "eva-auth",
        "eva-matrix"
    )
)

foreach ($repo in $Repos) {
    Write-Host "`nChecking $repo..." -ForegroundColor Cyan

    # Check README exists
    $readmePath = "../$repo/README.md"
    if (!(Test-Path $readmePath)) {
        Write-Host " Missing README.md" -ForegroundColor Red
        continue
    }

    # Validate README structure
    $readme = Get-Content $readmePath -Raw
    $requiredSections = @("Quick Start", "Architecture", "Development", "Documentation")

    foreach ($section in $requiredSections) {
        if ($readme -notmatch "##\s+$section") {
            Write-Host " Missing section: $section" -ForegroundColor Yellow
        }
    }

    # Check ADR directory
    $adrPath = "../$repo/docs/adr"
    if (!(Test-Path $adrPath)) {
        Write-Host " No ADR directory" -ForegroundColor Yellow
    } else {
        $adrCount = (Get-ChildItem $adrPath -Filter "*.md").Count
        Write-Host " ADR coverage: $adrCount decisions" -ForegroundColor Green
    }

    # Check bilingual compliance
    $readmeFrPath = "../$repo/README.fr.md"
    if (!(Test-Path $readmeFrPath)) {
        Write-Host " No French README" -ForegroundColor Yellow
    }
}

    Auto-Fix Workflows¶

    README Structure Fix¶

    # Auto-generate missing sections
param([string]$Repo)

$readme = Get-Content "../$Repo/README.md" -Raw

# Add missing Quick Start section
if ($readme -notmatch "## Quick Start") {
    $quickStart = @"

## Quick Start

### Prerequisites
- [Add prerequisites]

### Installation
\`\`\`bash
# Add installation steps
\`\`\`

### Usage
\`\`\`bash
# Add usage examples
\`\`\`
"@
    # Insert after first heading
    $readme = $readme -replace '(#[^\n]+\n)', "`$1$quickStart`n"
    $readme | Out-File "../$Repo/README.md" -Encoding utf8
    Write-Host " Added Quick Start section to $Repo" -ForegroundColor Green
}

    ADR Template Generation¶

    # Create new ADR from template
param(
    [string]$Repo,
    [string]$Title
)

# Get next ADR number
$adrPath = "../$Repo/docs/adr"
New-Item -Path $adrPath -ItemType Directory -Force | Out-Null

$existingAdrs = Get-ChildItem $adrPath -Filter "*.md"
$nextNumber = ($existingAdrs.Count + 1).ToString("D4")

$fileName = "$nextNumber-$($Title -replace '\s','-').md"
$template = Get-Content "templates/adr-template.md" -Raw
$template = $template -replace '\[Title\]', $Title
$template = $template -replace '\[NNNN\]', $nextNumber
$template = $template -replace '\[Date\]', (Get-Date -Format "yyyy-MM-dd")

$template | Out-File "$adrPath/$fileName" -Encoding utf8
Write-Host " Created ADR: $fileName" -ForegroundColor Green
