    Pattern P03: Scrum Agent¶

    Status: Draft Owner: POD-F (Foundation) Version: 1.0.0 Last Updated: 2025-12-12

    Overview¶

    The Scrum Agent automates stand-up outputs, runbook updates, and sprint board sync to maintain sprint cadence and reduce manual ceremony overhead. It ensures consistent documentation, accurate status tracking, and seamless sprint transitions.

    Purpose¶

    • Automate daily stand-up summary generation
    • Maintain sprint runbooks and ceremony artifacts
    • Sync GitHub sprint board with execution state
    • Enable transparent progress tracking across pods

    Automated Outputs¶

    1. Daily Stand-Up Summaries¶

    Location: docs/sprints/daily-standups/YYYY-MM-DD.md

    Auto-generated content:

    2. Sprint Board Sync¶

    Automation: - Pull GitHub issue states (Open, In Progress, In Review, Done) - Update project board columns - Flag stale issues (>72 hours without update) - Generate velocity metrics

    Sync frequency: Every 6 hours via GitHub Actions

    Workflow: .github/workflows/sprint-board-sync.yml

    4. Sprint Ceremonies Automation¶

    Sprint Planning¶

    • Pull backlog items tagged for sprint
    • Generate planning template with acceptance criteria
    • Create ceremony notes document

    Sprint Review¶

    • Compile completed stories
    • Generate demo checklist
    • Create stakeholder summary

    Sprint Retrospective¶

    • Aggregate velocity metrics
    • Identify patterns (blockers, cycle time)
    • Generate improvement suggestions

    Implementation¶

    Stand-Up Generator Script¶

    Location: tools/scrum/generate-standup.ps1

    # Generate daily stand-up summary
param(
    [string]$Sprint = (Get-Content ".sprint-marker" -ErrorAction SilentlyContinue)
)

$today = Get-Date -Format "yyyy-MM-dd"
$outputPath = "docs/sprints/daily-standups/$today.md"

# Query GitHub issues for current sprint
$issues = gh issue list --label "sprint-$Sprint" --json number,title,state,assignees,labels

# Parse yesterday's progress
$yesterday = (Get-Date).AddDays(-1).ToString("yyyy-MM-dd")
$yesterdayIssues = gh issue list --search "updated:>=$yesterday" --label "sprint-$Sprint" --json number,title,state

# Generate summary
$summary = @"
# Daily Stand-Up - $today

**Sprint**: Sprint $Sprint
**Generated**: $(Get-Date -Format "yyyy-MM-dd HH:mm") UTC

## Yesterday's Progress
"@

foreach ($issue in $yesterdayIssues) {
    $status = if ($issue.state -eq "closed") { "‚úÖ" } else { "üîÑ" }
    $summary += "`n- $status Issue #$($issue.number): $($issue.title)"
}

# Add today's plan, blockers, velocity sections...

$summary | Out-File -FilePath $outputPath -Encoding utf8
Write-Host "‚úÖ Stand-up summary: $outputPath" -ForegroundColor Green

    Runbook Update Script¶

    Location: tools/scrum/update-runbook.ps1

    param(
    [string]$Sprint = (Get-Content ".sprint-marker" -ErrorAction SilentlyContinue)
)

$runbookPath = "docs/sprints/sprint-$Sprint/runbook.md"

# Query current sprint issues
$issues = gh issue list --label "sprint-$Sprint" --json number,title,state,assignees,labels,comments

# Build progress tracker table
$progressTable = "| Story | Status | Owner | Progress | Blockers |`n"
$progressTable += "|-------|--------|-------|----------|----------|`n"

foreach ($issue in $issues) {
    $status = switch ($issue.state) {
        "closed" { "‚úÖ Done" }
        "open" {
            if ($issue.labels -contains "in-progress") { "üîÑ In Progress" }
            elseif ($issue.labels -contains "blocked") { "üöß Blocked" }
            else { "üìã Ready" }
        }
    }

    $owner = $issue.assignees[0].login ?? "Unassigned"
    $progress = "TBD"  # Parse from comments or labels
    $blockers = if ($issue.labels -contains "blocked") { "See issue" } else { "None" }

    $progressTable += "| #$($issue.number) | $status | $owner | $progress | $blockers |`n"
}

# Update runbook (replace progress tracker section)
$runbook = Get-Content $runbookPath -Raw
$runbook = $runbook -replace '(?s)(\| Story \| Status.*?\n)(\|.*?\n)+', $progressTable

$runbook | Out-File -FilePath $runbookPath -Encoding utf8
Write-Host "‚úÖ Updated: $runbookPath" -ForegroundColor Green
