    Pattern P05: Scaffolder Agent¶

    Status: Draft Owner: POD-F (Foundation) Version: 1.0.0 Last Updated: 2025-12-12

    Overview¶

    The Scaffolder Agent provides a CLI tool that bootstraps EVA repository layout, documentation structure, and configuration files following established patterns and best practices.

    Purpose¶

    • Standardize repository structure across EVA Suite
    • Accelerate new repo creation with proven templates
    • Ensure consistency in configuration and documentation
    • Reduce setup time from hours to minutes

    Scaffold Command¶

    CLI Interface¶

    # Create new EVA repository
eva-scaffold create <repo-name> --type <template> [options]

# Templates
--type api          # API service (Node.js/TypeScript)
--type ui           # UI application (React/Vue)
--type lib          # Shared library
--type infra        # Infrastructure as code
--type pipeline     # Data pipeline

# Options
--pod <POD-X>       # Assign to POD
--owner <pattern>   # Assign owner pattern
--bilingual         # Include FR-CA docs
--gc-design         # Include GC Design System

    Usage Examples¶

    # Create new API service
eva-scaffold create eva-notifications --type api --pod POD-F --bilingual

# Create UI app with GC Design System
eva-scaffold create eva-admin-ui --type ui --pod POD-X --gc-design --bilingual

# Create shared library
eva-scaffold create eva-utils --type lib --owner P04-LIB

    Smoke Test¶

    Automated Validation¶

    Script: tools/scaffolder/smoke-test.ps1

    param([string]$RepoPath)

Write-Host "Running scaffold smoke test..." -ForegroundColor Cyan

# Check required files exist
$requiredFiles = @(
    "README.md",
    "package.json",
    ".github/workflows/ci.yml",
    "docs/adr/0001-initial-architecture.md"
)

foreach ($file in $requiredFiles) {
    if (!(Test-Path "$RepoPath/$file")) {
        Write-Host "❌ Missing: $file" -ForegroundColor Red
        return $false
    }
}

# Validate package.json
$pkg = Get-Content "$RepoPath/package.json" | ConvertFrom-Json
if (!$pkg.scripts.test) {
    Write-Host "❌ Missing test script in package.json" -ForegroundColor Red
    return $false
}

# Run install
Push-Location $RepoPath
npm install --silent
if ($LASTEXITCODE -ne 0) {
    Write-Host "❌ npm install failed" -ForegroundColor Red
    Pop-Location
    return $false
}

# Run lint
npm run lint --silent
if ($LASTEXITCODE -ne 0) {
    Write-Host "❌ Lint failed" -ForegroundColor Red
    Pop-Location
    return $false
}

# Run tests
npm test --silent
if ($LASTEXITCODE -ne 0) {
    Write-Host "❌ Tests failed" -ForegroundColor Red
    Pop-Location
    return $false
}

Pop-Location
Write-Host "✅ Smoke test passed" -ForegroundColor Green
return $true
